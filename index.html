Texture Segmentation

This LBP based unsupervised grayscale texture based segmentation algorithm was implemented as a part of Digital Image Processing project to compare 2 texture segmentation algorithms. The other algorithm used was Efficient graph based segmentation.

This research paper is available here.

This algorithm implementation was ranked first on prague evaluator in grayscale category. (as on 26 December, 2012).

The full report and presentation is available.

To run the code (in matlab)

cd TextureSegmentationLBP/src;
addpath('test');
I = imread('image_file_name.jpg');
[S,R,Q] = runIt(I); % runIt implements the full pipeline
imshow(R,[]); %to see the segmentation
